# Python 基础笔记

## 变量与数据类型

Python 是动态类型语言，变量不需要声明类型：

```python
name = "Fujiko"
age = 22
pi = 3.14159
is_student = True
```

## 列表与字典

```python
# 列表
fruits = ["苹果", "香蕉", "橘子"]
fruits.append("葡萄")

# 字典
person = {
    "name": "Fujiko",
    "age": 22,
    "skills": ["Python", "JavaScript"]
}
```

## 函数

```python
def greet(name):
    return f"你好，{name}！"

print(greet("世界"))
```
