## 关系型数据库

```
表格形式存储 支持SQL查询
适用：交易系统 ERP
```

## 非关系型数据库（）

### 文档型数据库 MongoDB

```
数据模型：类似JSON  无固定表结构
适用：内容管理 实时分析 日志存储
```

### 键值型数据库redis

```
数据模型：key-value 内存优先 支持持久化
适用：缓存 会话管理 实时排行榜
```

## 时间序列数据库

### influxDB

```
数据模型：时间戳组织数据 自动压缩过期数据
适用场景：服务监控 传感器数据采集
```

## 图数据库

### Ne04j

```
数据模型：节点和关系表示数据
适用场景：社交网络分析 欺诈检测 推荐系统
```
