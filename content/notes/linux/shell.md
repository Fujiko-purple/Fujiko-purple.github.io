## 基本语法

```shell
第一步：新建脚本文件
vim 文件名.sh

第二步:定义脚本解释器 注释说明
#!/bin/bash

第三步；编写脚本内容

第四步：添加权限
chmod +x 文件名.sh
#添加执行权限

第五步：运行脚本
./脚本名字

bash 脚本名字	#不需要权限也可以运行
```

## 基础用法

### 使用echo命令

```shell
#基本输出功能
echo -e		#控制格式
\n		#换行

#输出重定向
	标准重定向
		输出的内容 > 输入的文件
			#会将输出的内容覆盖掉输入的文件内容
	追加重定向
		输出的内容 >> 输入的文件
			#在输入文件原内容末尾添加新内容
	错误输出重定向
		2>
			#将输出的错误内容重定向到文件
	同时重定向标准输出和错误输出
		&>
			#会将错误的正确的信息同时输出重定向
```

### 变量的使用

```shell
#变量赋值
var=value

#变量使用方法 ： 在变量名前加上$
ehco $var
#输出value

#位置参数
$1
$2
$3
bash demo.sh 陈雷 阿彪 阿丽

这是第一个变量 陈雷
这是第二个变量 阿彪
这是第三个变量 阿丽

#特殊变量
echo $?
	表示上一个命令的退出状态
	0通常表示成功
	非0通常表示失败
```
