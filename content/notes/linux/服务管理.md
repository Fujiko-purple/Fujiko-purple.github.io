## systemd

```shell
#服务启动停止

#启动	
systemctl start 服务名

#停止
systemctl stop 服务名

#重启
systemctl restart 服务名

#重载
systemctl roload 服务名
```



## 查看服务日志

```shell
journalctl

journalctl -u 服务名

journactl -f 	#用于监控实时日记

journalctl --since "开始时间" --until "结束时间"	#特定时间段日记
```
