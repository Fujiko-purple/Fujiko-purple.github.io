## useradd

```shell
#创建用户 默认添加到/home

#密码设置 
passwd 用户名	#修改用户密码
passwd -l 用户名	#锁定账号 让账号无法登录
passwd -u 用户名	#解锁用户 让用户可以重新登录
passwd -e 用户名	#强制用户下次登录修改密码

chage #设置密码过期时间
chage -l	#显示密码时间相关信息
chage -E 时间 账户名	#设置密码过期时间
```



## 用户组管理 groupadd

```shell
#创建组
groupadd 组名

#将用户添加到组当中
usermod -aG	组名 用户名

#将用户从组中踢出
gpasswd -d 用户名 组名

#删除组
groupdel 组名

#查看组内多少成员
getent group 组名

#文件下存储的组
/etc/group

```
