## 磁盘命名



## 查看磁盘信息

```shell
df
df -h	#显示多少g
du -sh 目录	#查看占多少大小
lsblk	#查看文件分区挂载信息
iostat	#查看磁盘i/o统计
```



## 创建新分区

```shell
#分区操作
手动添加

#选择分区类型
fdisk

#分区大小规划

#文件格式化
mkfs.ext4

#挂载
创建目录
mkdir 目录	

mount 磁盘 挂载点（目录）

进行卸载时
umount 目录

#自动挂载配置
fstab
vim /etc/fstab	#编写内容

mount -a 
```
