## ls

```shell
ls -l
ls -a
ls -r	#反转排序顺序
ls -t 	#按时间排序
ls -F	#文件类型都会显示
```

## cd

```shell
pwd
cd -	#返回上一次目录
cd ~	#回到家目录
cd /	#回到根目录
cd ..	#回到上一级
cd		#回到/root
```

## mkdir

```shell
mkdir-创建目录
mkdir -p	#创造多级目录
mkdir -m	#创建时设置权限
mkdir -v	#输出显示目录是否创建成功
```



## rmdir

```shell
rmdir-删除文件或目录
#只能删除目录
rmdir -p 	#删除多级目录 前提是每个目录内容为空 
```

## rm

```shell
rm -r	#删除目录
rm -f	#强制删除某个文件
rm -rf	#强制删除非空目录不提示确认
```

## cp

```shell
#复制文件
cp -r
```

## mv

```shell
#移动文件到指定目录重命名文件
mv 原文件 改名文件
mv 文件 移动目录
mv 原文件 移动目录/改名文件

mv 原文件 被覆盖文件	#注意 mv不仅是覆盖 并且还是改名 所以原文件覆盖另一个文件后 名字改成另一个文件

mv -i #提示是否覆盖
mv -v #显示移动过程 包括改名
#iv可一起使用
```

## touch

```shell
#创建空文件
可一次性创建多个文件
touch -t 设置的时间戳 文件	#设置特定时间戳
touch 文件名	#恢复时间
touch -t 时间 *文件类型	#根据通配符统一改变文件时间戳
touch --help	
```

## cat

```shell
#查看文件内容
cat -E 文件名	#显示不会打印的内容 一般是每行最后都会有$
cat 文件1 文件2 > 文件3	#合并多个文件 合并的内容会覆盖文件3 文件内容顺序根据文件顺序排列
```

## echo

```shell
echo 输出的内容

#输出定义的变量
变量名=变量
echo $变量名
#会输出变量
#在shell中会使用到

echo $(cat 文件名)
#输出文件内容 并且不会按照文件内容换行 只会空格
```
